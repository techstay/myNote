# kotlin

我的 Kotlin 学习代码

## 开发环境

安装 Jetbrains Intellij IDEA，这样就齐全了。

## 代码

- kotlin-demo，kotlin 学习代码
- jetbrains-quest，Jetbrains 谜题的解密方法，包含了全部 3 个谜题的解密函数。
- kotlin koans，kotlin 练习，用来演练 kotlin 的各种常用功能
