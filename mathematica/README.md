# mathematica

## 环境配置

有条件的直接使用 Mathematica 作为前端，或者也可以安装 Mathematica Engine 然后使用 vscode 等配合。

使用 Mathematica 的时候，点击*面板->格式化代码*就可以格式化 Mathematica 代码了。
